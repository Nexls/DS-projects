# Учебные проекты

**В этом репозитории собраны проекты, которые я выполнял в рамках курса по Data Science от Яндекс.Практикума.
Проекты выполнены в тетрадках Jupyter Notebook.**

В папках с проектами имеются файлы *DESCRIPTION.md*, в которых содержится информация о проекте:

- Название
- Краткое описание проекта
- Поставленная задача
- Описание данных
- Использованные библиотеки
